## Покрокова інструкція

### 1. Згенеруйте сайт через AI

Використовуйте Claude (Artifacts) або ChatGPT із промптом із розділу [Промпти](/sites-25min/prompts). На виході — один файл `index.html` з вбудованими стилями.

### 2. Створіть репозиторій

1. Зайдіть на `github.com` → New repository.
2. Назва: `client-name-site`.
3. Public → Create.
4. Натисніть **Add file → Upload files**, перетягніть `index.html` і зображення.

### 3. Підключіть Cloudflare Pages

1. Реєстрація на `pages.cloudflare.com` (безкоштовно).
2. **Create project → Connect to Git → GitHub** → виберіть репозиторій.
3. Build command: залиште порожнім. Output directory: `/`.
4. **Save and Deploy** — за 30 секунд сайт живий на `client-name.pages.dev`.

### 4. Підключіть власний домен

- Купіть домен на Cloudflare Registrar (від \$8/рік — оплачує клієнт).
- Pages → Custom domains → Add → введіть домен.
- SSL і DNS налаштовуються автоматично.

## Альтернативи хостингу

| Платформа | Банери | Власний домен | Ліміти |
| --- | --- | --- | --- |
| Cloudflare Pages | Немає | Так, безкоштовно | Необмежений трафік |
| Vercel | Немає (Hobby) | Так | 100 ГБ/міс |
| Netlify | Немає | Так | 100 ГБ/міс |
| GitHub Pages | Немає | Так | 100 ГБ/міс |

<Warning>
  Не використовуйте безкоштовні плани Wix / Tilda / WordPress.com — у них банери та реклама на сайті клієнта.
</Warning>

## Як продавати «безкоштовний» сайт

Клієнту не кажете «безкоштовно». Продаєте **результат**: «Сучасний сайт на власному домені з SSL і швидким хостингом — \$400».

Ваші витрати: $0 + $8 за домен (оплачує клієнт).
