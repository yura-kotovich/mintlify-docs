# Промпти для сайтів за 25 хвилин

Готові промпти, які перетворюють картку Google Maps на готовий сайт.

<Note>
  Найкраще працюють у **Claude 3.5 Sonnet (Artifacts)** і **ChatGPT-4o з Canvas**. Обидва віддають готовий HTML, який одразу можна публікувати.
</Note>

## Промпт №1. Бриф із Google Maps

Копіюєте все, що бачите на сторінці бізнесу в Maps, і вставляєте в цей промпт:

```text
Ти — senior веб-дизайнер. На основі даних бізнесу нижче склади структурований бриф для односторінкового сайту:

[вставити: назва, опис, послуги/меню, фото, відгуки, адреса, телефон, години роботи]

Поверни:
1. УТП у 1 реченні
2. Структура з 6–8 секцій
3. Tone of voice (3 прикметники)
4. Кольорова палітра (5 hex)
5. Шрифтова пара (Google Fonts)
```

## Промпт №2. Генерація сайту

```text
Згенеруй односторінковий сайт для [тип бізнесу] на основі брифу нижче.

Вимоги:
- Один файл index.html з вбудованими CSS і JS
- TailwindCSS через CDN
- Адаптивний дизайн (mobile-first)
- Сучасний стиль 2025: великі типографічні заголовки, м'які тіні, плавні анімації при скролі
- Секції: Hero, Про нас, Послуги/Меню, Галерея, Відгуки, Контакти, Футер
- Sticky навігація, smooth scroll до секцій
- CTA-кнопки: "Забронювати" / "Зателефонувати" — клікабельний tel:
- Lighthouse Performance > 90
- Без зовнішніх залежностей, окрім Tailwind CDN і Google Fonts

Бриф:
[вставити вивід Промпту №1]
```

## Промпт №3. Поліровка

Коли отримали чорновик:

```text
Покращ цей сайт:
1. Зроби Hero більш ефектним (градієнт + фонове фото з overlay)
2. Додай мікроанімації на hover для карток
3. Додай секцію FAQ з accordion
4. Зроби форму контактів із валідацією
5. Додай OpenGraph і Twitter Card meta-теги
6. SEO: title, description, schema.org LocalBusiness

Поверни оновлений index.html повністю.
```

## Промпт №4. Переклад

```text
Переклади весь текстовий контент сайту на [англійську / іспанську / німецьку]. 
Збережи структуру HTML без змін. Адаптуй CTA під локальну культуру.
```

## Промпт №5. Демо-повідомлення власнику

```text
Напиши коротке (до 400 символів) повідомлення власнику бізнесу [назва] в Instagram DM.

Контекст: я зробив демо нового сайту для його закладу, ось посилання: [URL].

Тон: дружній, без "продажності", акцент на тому, що це готове демо, яке можна запустити за добу. Закінчи відкритим питанням.
```
