Ця колекція містить перевірені промпти, які можна копіювати та одразу використовувати у ChatGPT, Claude чи будь-якому іншому ШІ-інструменті. Кожен запит структурований так, щоб давати корисний і конкретний результат без додаткового налаштування. Щоб адаптувати промпт під свою ситуацію, просто замініть виділені деталі — назву компанії, тему, аудиторію — на власні дані.

## Бізнес-комунікація

<Tabs>
  <Tab title="Email">
    **Холодний лист потенційному клієнту**

    ```text
    Напиши холодний email потенційному клієнту від імені B2B-компанії.

    Контекст:
    - Ми надаємо послуги бухгалтерського аутсорсингу
    - Отримувач: власник невеликого інтернет-магазину
    - Мета листа: домовитися про 20-хвилинний дзвінок

    Вимоги:
    - Тема листа: інтригуюча, без спаму
    - Обсяг: не більше 120 слів
    - Тон: дружній, але професійний
    - Закінчити конкретним закликом до дії з простим запитанням
    ```

    ---

    **Лист із вибаченням за затримку**

    ```text
    Напиши вибачення клієнту за затримку доставки.

    Ситуація: замовлення мало бути доставлено 3 дні тому, затримка через
    проблеми з логістичним партнером. Клієнт написав скаргу.

    Структура листа:
    1. Щире вибачення без виправдань
    2. Коротке пояснення (без перекладання провини)
    3. Що ми вже зробили для вирішення
    4. Конкретна нова дата доставки
    5. Компенсація (знижка 10% на наступне замовлення)

    Тон: теплий, відповідальний, клієнтоорієнтований.
    ```

    ---

    **Follow-up після зустрічі**

    ```text
    Напиши follow-up лист після ділової зустрічі.

    Деталі зустрічі:
    - Обговорювали потенційну співпрацю з дистриб'ютором
    - Домовилися надіслати комерційну пропозицію протягом тижня
    - Наступний крок: дзвінок через 10 днів

    Лист має:
    - Подякувати за час та продуктивну розмову
    - Стисло підсумувати ключові домовленості (2–3 пункти)
    - Підтвердити свої наступні дії з дедлайном
    - Запропонувати конкретний час для наступного дзвінка

    Обсяг: до 150 слів. Тон: ділової, але людяний.
    ```
  </Tab>

  <Tab title="Презентації">
    **Структура презентації для інвестора**

    ```text
    Створи структуру pitch-deck для стартапу.

    Про компанію: мобільний застосунок для планування харчування та доставки
    продуктів, цільова аудиторія — зайняті міські жителі 25–40 років.

    Розроби структуру з 10 слайдів:
    - Для кожного слайду вкажи: назву, головну думку (1 речення),
      що саме показати (дані, графік, зображення)
    - Виділи 2–3 ключові метрики, які найбільше зацікавлять інвестора
    - Вкажи, на якому слайді розмістити «гачок» для уваги аудиторії
    ```

    ---

    **Вступний слайд та ключові меседжі**

    ```text
    Напиши текст для вступного слайду та 3 ключові меседжі презентації.

    Тема: Звіт про результати маркетингової кампанії за квартал
    Аудиторія: топ-менеджмент компанії
    Головний результат: перевиконання плану на 23%

    Для вступного слайду: заголовок + один підзаголовок (не більше 15 слів разом)

    Для ключових меседжів: три речення, кожне починається з сильного дієслова,
    містить конкретну цифру і відповідає на питання «що це означає для бізнесу».
    ```

    ---

    **Заключна частина з закликом до дії**

    ```text
    Напиши заключну частину презентації з чітким закликом до дії.

    Контекст: презентуємо нову CRM-систему команді продажів (15 осіб)
    Мета: отримати згоду на пілотний запуск протягом місяця

    Заключна частина має включати:
    1. Коротке резюме трьох головних переваг (по одному реченню кожна)
    2. Чіткий наступний крок: що потрібно зробити, хто відповідальний, коли
    3. Відповідь на можливе заперечення «у нас немає часу на навчання»
    4. Мотивуючий фінальний меседж

    Стиль: впевнений, позитивний, без тиску.
    ```
  </Tab>

  <Tab title="Звіти">
    **Щотижневий звіт для керівника**

    ```text
    Допоможи написати щотижневий звіт для керівника.

    Мої результати за тиждень:
    - Провів 8 зустрічей з клієнтами
    - Закрив 2 угоди на загальну суму 45 000 грн
    - Підготував 5 комерційних пропозицій
    - Не встиг: оновити базу даних (перенесено на наступний тиждень)

    Оформи у форматі:
    ✅ Виконано (з ключовими цифрами)
    📊 Результати (вплив на бізнес-показники)
    ⚠️ Відкриті питання / що перенесено
    🎯 План на наступний тиждень (3 пріоритети)

    Тон: лаконічний, фактологічний, без зайвих слів.
    ```

    ---

    **Аналітичний звіт про конкурентів**

    ```text
    Склади шаблон аналітичного звіту про конкурентів для внутрішнього використання.

    Ринок: онлайн-освіта (курси для дорослих)
    Кількість конкурентів для аналізу: 3

    Для кожного конкурента структуруй аналіз за блоками:
    1. Загальна інформація (розмір, позиціонування, ЦА)
    2. Продуктова лінійка та ціни
    3. Сильні сторони
    4. Слабкі сторони та прогалини
    5. Маркетингові канали та меседжі

    В кінці: порівняльна таблиця та 3 висновки для нашої стратегії.
    ```
  </Tab>

## Маркетинг та контент

<Accordion title="Пости для соцмереж">
  **Пост для LinkedIn (експертний контент)**

  ```text
  Напиши пост для LinkedIn на тему особистої продуктивності.

  Формат посту:
  - Перший рядок: несподіваний або провокаційний факт/твердження (гачок)
  - 2–3 абзаци: розкриття теми з особистим досвідом або прикладом
  - Практичний висновок або порада (1–2 речення)
  - Питання до аудиторії для залучення коментарів
  - 3–5 хештегів

  Тон: автентичний, розмовний, без корпоративного пафосу.
  Обсяг: 150–200 слів.
  ```

  ---

  **Серія постів для Instagram (3 пости)**

  ```text
  Створи серію з 3 постів для Instagram для просування онлайн-курсу з Excel.

  Цільова аудиторія: офісні працівники 22–35 років, хочуть підвищити кваліфікацію

  Пост 1 — Проблема: покажи «біль» аудиторії (витрати часу на ручну роботу)
  Пост 2 — Рішення: що зміниться після курсу (конкретні результати)
  Пост 3 — Соціальний доказ: формат відгуку/кейсу студента

  Для кожного посту:
  - Текст до 100 слів
  - Опис зображення або відео
  - 5 хештегів (мікс широких і нішевих)
  ```
</Accordion>

<Accordion title="SEO-тексти">
  **SEO-стаття для блогу**

  ```text
  Напиши SEO-оптимізовану статтю для блогу компанії.

  Тема: «Як вибрати CRM-систему для малого бізнесу»
  Ключові слова: CRM для малого бізнесу, вибір CRM, впровадження CRM
  Обсяг: 800–1000 слів
  Аудиторія: власники малого бізнесу без технічних знань

  Структура:
  - H1: заголовок з ключовим словом
  - Вступ: проблема та обіцянка (100 слів)
  - 4–5 підзаголовків H2 з практичним змістом
  - Список критеріїв вибору (маркований список)
  - Порівняльна таблиця 3–4 рішень
  - Висновок + заклик до дії

  Стиль: експертний, але доступний. Без технічного жаргону.
  ```

  ---

  **Мета-опис для сторінки сайту**

  ```text
  Напиши 5 варіантів мета-опису для сторінки послуги.

  Послуга: розробка мобільних застосунків
  Ключове слово: розробка мобільних додатків
  Вимоги:
  - Довжина: 150–160 символів кожен
  - Містить ключове слово
  - Є заклик до дії
  - Підкреслює унікальну перевагу (швидко, надійно, під ключ)

  Пронумеруй варіанти та вкажи кількість символів після кожного.
  ```
</Accordion>

  **Рекламний пост для Facebook/Instagram**

  ```text
  Напиши рекламний пост для таргетованої реклами у Facebook.

  Продукт: доставка готової їжі на тиждень (раціон для схуднення)
  Аудиторія: жінки 28–45 років, стежать за здоров'ям, зайняті
  Мета: перехід на сайт та оформлення першого замовлення
  Спеціальна пропозиція: -20% на перший тиждень

  Структура:
  - Гачок: перші 2 рядки, що зупиняють скролінг
  - Основний текст: проблема → рішення → переваги (3–4 речення)
  - Заклик до дії: одне чітке речення

  Обсяг: до 125 слів. Тон: дружній, мотивуючий.
  ```
</Accordion>

## Навчання та дослідження

Ці промпти допоможуть швидко освоїти нову тему, систематизувати знання або підготуватися до іспиту чи презентації.

```text
Поясни мені [ТЕМА] так, ніби я повний новачок.

Тема: блокчейн і криптовалюти

Структура пояснення:
1. Проста аналогія з повсякденного життя (2–3 речення)
2. Як це працює — 5 кроків простою мовою
3. Три реальних приклади використання сьогодні
4. Головні переваги та ризики (по 3 пункти)
5. З чого почати, якщо хочу розібратися глибше (2–3 ресурси або кроки)

Уникай технічного жаргону. Якщо використовуєш термін — одразу пояснюй його.
```

---

```text
Допоможи підготуватися до іспиту / співбесіди.

Тема: основи маркетингу (для студента економічного факультету)
Формат іспиту: усна відповідь, 20 хвилин

Зроби наступне:
1. Визнач 10 найважливіших концепцій, які точно запитають
2. Для кожної концепції: коротке визначення (1–2 речення) + приклад
3. Склади 5 типових питань із зразковими відповідями
4. Дай 3 поради, як краще структурувати відповідь на усному іспиті
```

---

```text
Зроби огляд досліджень на тему.

Тема: вплив короткого сну на продуктивність праці

Структура огляду:
- Що кажуть дослідження: 5 ключових знахідок з джерелами (або приблизними посиланнями)
- Консенсус vs суперечності в науковій спільноті
- Практичні висновки для звичайної людини
- Що ще не досліджено або залишається відкритим питанням

Рівень: зрозумілий для людини без наукового ступеня, але з деталями для глибшого розуміння.
```

## Особиста продуктивність

Використовуйте ці промпти для планування, узагальнення інформації та генерації ідей.

```text
Допоможи спланувати мій день.

Мої задачі на сьогодні:
- Написати звіт (терміново, дедлайн до 15:00)
- Відповісти на 12 листів у пошті
- Зустріч з командою о 11:00 (1 година)
- Підготувати презентацію на п'ятницю
- Зателефонувати 3 клієнтам
- Переглянути контракт

Мій робочий день: 9:00 — 18:00, обід 13:00–14:00.
Я найбільш продуктивний у першій половині дня.

Склади розклад дня з часовими блоками, враховуючи пріоритети та мою продуктивність.
Додай 2–3 поради, як уникнути прокрастинації сьогодні.
```

---

```text
Зроби короткий конспект наступного тексту.

[ВСТАВТЕ ТЕКСТ АБО ПОСИЛАННЯ]

Формат конспекту:
- Головна ідея: 1–2 речення
- Ключові тези: маркований список (5–7 пунктів)
- Важливі факти або цифри: окремий список
- Висновок або практичне застосування: 2–3 речення

Рівень деталізації: достатній, щоб зрозуміти суть без читання оригіналу.
```

---

```text
Проведи мозковий штурм ідей.

Задача: придумати ідеї для корпоративного тімбілдингу
Обмеження: бюджет до 500 грн на людину, 20 учасників, онлайн-формат, 3 години

Згенеруй 15 різноманітних ідей.
Після списку ідей:
- Вибери ТОП-3 з поясненням, чому вони найкращі
- Для кожної з ТОП-3 вкажи: що потрібно підготувати, хто веде, орієнтовна програма
```

---

<Note>
Ці промпти — відправна точка, а не фінальний варіант. Додайте конкретні деталі вашої ситуації: назву компанії, галузь, цільову аудиторію, тон спілкування — і результат стане ще більш точним і корисним.
</Note>
